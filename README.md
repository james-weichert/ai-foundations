# An Introduction to the Foundations and Impacts of Artificial Intelligence

_UW Allen School Admitted Students Day 2026_

[**James Weichert**](https://jpw.info/), Assistant Teaching Professor

In this lecture, we’ll design a simple algorithm to recommend Seattle restaurants based on user preferences. This algorithm is a rudimentary precursor to modern artificial intelligence (AI) systems, which use data to independently generate outputs or make decisions. By exploring a simplified example, we’ll discuss how human decisions affect the outputs of AI models, and how emerging AI systems are impacting society.
